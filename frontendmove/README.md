# SUI Merch Store

The official merch store for the SUI community.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Sui SDK

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Purchase Flow

This store supports purchasing SUI merchandise directly with SUI tokens.
